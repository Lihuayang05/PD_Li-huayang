Book Recommendation Project

This web app provides book recommendations by randomly generating a book suggestion at the click of a button. The platform features a clean and user-friendly design.

  • Randomized book recommendation functionality
  • User authentication system with registration, login, and logout
  • CRUD features for managing books (add, edit, delete)
  • Integrated MySQL database to store user information and book details
  • Dynamic UI using JavaScript for seamless user interaction
